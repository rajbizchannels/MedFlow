# AureonCare Logo

Please save the AureonCare logo image as `aureoncare-logo.png` in this directory.

The logo should be the image with:
- The golden network icon
- "AUREON CARE" text (AUREON in gold, CARE in blue)
- "Empowering Healthcare Solutions" tagline

The header has been configured to display this logo at a height of 48px (h-12) with automatic width scaling.

Recommended image specifications:
- Format: PNG (with transparent background preferred)
- Dimensions: At least 200px in height for optimal quality
- Aspect ratio: Maintain original logo proportions

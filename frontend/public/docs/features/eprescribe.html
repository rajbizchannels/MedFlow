Electronic Prescribing (ePrescribe)

üÜï New in Version 1.1:

The ePrescribe modal now features edit mode with prefilled medication names, step-by-step workflow navigation, and direct linking to diagnoses!

AureonCare's ePrescribe module enables providers to electronically prescribe medications, check drug interactions, and send prescriptions directly to pharmacies. The system is integrated with national drug databases and pharmacy networks.

Accessing ePrescribe

From Patient Chart

  1. Open a patient's chart
  2. Go to the "Medications" tab
  3. Click "+ New Prescription" button
  4. The ePrescribe modal opens with a 4-step workflow

From Clinical Encounter

  1. During a patient visit or clinical note
  2. Click "Prescribe Medication" in the encounter toolbar
  3. The ePrescribe modal opens with patient context pre-loaded

ePrescribe Workflow (4 Steps)

The ePrescribe modal guides you through a step-by-step process:

Step 1: Search Medication

  1. Search for Drug: Type medication name or NDC code
  2. Select from Results: Choose the specific drug formulation
    • Brand name or generic
    • Strength (e.g., 10mg, 20mg)
    • Form (tablet, capsule, liquid, etc.)
  3. View Drug Information: Review indication, warnings, and contraindications
  4. Click "Next" to proceed to prescription details
üí° Edit Mode Enhancement:

When editing an existing prescription, Step 1 now automatically prefills the medication name, allowing you to quickly update dosage or instructions without re-searching!

Step 2: Prescription Details

Enter detailed prescription information:

Field Description Example
Dosage Strength and quantity per dose 10mg, 1 tablet
Frequency How often to take medication Once daily, Twice daily, Every 6 hours
Route Administration method Oral, Topical, IV, IM, etc.
Duration Length of treatment 7 days, 30 days, Until finished
Quantity Total number to dispense 30 tablets, 100ml
Refills Number of refills allowed 0, 1, 2, 3, PRN
Instructions Patient-facing directions (SIG) "Take 1 tablet by mouth daily with food"
Notes Internal clinical notes "For hypertension management"

Special Options

  • Generic Substitution: Toggle to allow/disallow generic substitution ("DAW" - Dispense As Written)
  • Prior Authorization: Check if prior auth is required
  • Controlled Substance: System flags Schedule II-V medications automatically

Step 3: Select Pharmacy

  1. Patient's Preferred Pharmacy: Pre-selected if on file
  2. Search for Pharmacy: Find by name, address, or zip code
  3. Recent Pharmacies: Choose from recently used pharmacies
  4. Add New Pharmacy: Add a pharmacy to the system if not found

Selected pharmacy must support electronic prescribing (most major chains do).

‚ö†Ô∏è Controlled Substances:

Schedule II medications may require paper prescription in some states. Check your state regulations. AureonCare will alert you if electronic prescribing is not allowed.

Step 4: Review and Submit

  1. Review All Information:
    • Patient demographics and allergies
    • Medication details and instructions
    • Pharmacy information
    • Drug interactions and warnings
  2. Link to Diagnosis (New!):
    • Select primary diagnosis for this prescription
    • Links prescription to ICD-10 code for insurance
    • Improves claim approval rates
  3. Digital Signature: Apply your electronic signature
  4. Click "Submit Prescription"

Editing Existing Prescriptions

Edit vs. New Prescription

To modify an active prescription:

  1. Go to Patient Chart > Medications
  2. Find the prescription in the active medications list
  3. Click "Edit" icon
  4. The ePrescribe modal opens in Edit Mode:
    • Medication name is prefilled (can't be changed - create new prescription to change drug)
    • All current details are pre-populated
    • You can navigate between steps while maintaining context
    • Update dosage, frequency, quantity, or instructions
  5. Click "Update Prescription"
  6. System sends updated prescription to pharmacy
üìù Version History:

All prescription changes are tracked in the medication history. Click "View History" to see all versions and who made changes.

Managing Prescriptions

Renewing a Prescription

  1. Click "Renew" next to an active or expired prescription
  2. Review and update details if needed
  3. Update refill count and quantity
  4. Submit renewed prescription

Discontinuing a Prescription

  1. Click "Discontinue"
  2. Select reason:
    • Treatment completed
    • Adverse reaction
    • No longer needed
    • Changed to different medication
    • Other (specify)
  3. Add optional notes
  4. Confirm discontinuation

Discontinued medications move to the "Past Medications" section but remain in the patient's history.

Linking Prescriptions to Diagnoses

New in v1.1: You can now link each prescription to a diagnosis code.

Why Link Diagnoses?

  • Insurance Requirements: Many payers require diagnosis justification
  • Clinical Documentation: Creates clear link between problem and treatment
  • Improved Claims: Reduces claim denials and prior auth rejections
  • Better Analytics: Track prescribing patterns by condition

How to Link

  1. In Step 4 (Review), look for "Link to Diagnosis" section
  2. Click "Select Diagnosis"
  3. Choose from:
    • Diagnoses from current encounter
    • Active diagnoses in patient's problem list
    • Search for new ICD-10 code
  4. Selected diagnosis appears on prescription
üí° Quick Tip:

When prescribing from within a clinical note, diagnoses from that encounter are pre-populated for easy selection.

Controlled Substances

DEA Requirements

To prescribe controlled substances electronically:

  • Provider must have valid DEA number on file
  • Two-factor authentication required for each CS prescription
  • State-specific rules apply (some states require paper for Schedule II)
  • Refills not allowed for Schedule II

EPCS Authentication

  1. Complete prescription as normal
  2. When submitting a controlled substance prescription:
  3. System prompts for additional authentication:
    • Password confirmation
    • SMS code to registered phone
    • Or biometric verification
  4. Enter authentication credentials
  5. Prescription is digitally signed and transmitted

Pharmacy Management

Adding a New Pharmacy

  1. From the pharmacy selection step, click "+ Add Pharmacy"
  2. Search NCPDP database by:
    • Pharmacy name
    • Address or zip code
    • NCPDP number
  3. Select pharmacy from results
  4. Verify ePrescribe capability (shown in listing)
  5. Save to patient's preferred pharmacies

Setting Patient's Preferred Pharmacy

  1. Go to Patient Chart > Demographics
  2. Click "Edit Contact Information"
  3. Select "Preferred Pharmacy" from dropdown
  4. Save changes

This pharmacy will be pre-selected for all future prescriptions for this patient.

Prescription History

View complete prescription history:

  • Active Medications: Currently prescribed and active
  • Past Medications: Discontinued or completed
  • Medication History: All prescriptions ever written

Filter by:

  • Date range
  • Prescribing provider
  • Medication name or class
  • Status (active, discontinued, etc.)

Troubleshooting

Prescription Failed to Send

  • Check internet connection
  • Verify pharmacy supports ePrescribe
  • Ensure pharmacy information is complete and correct
  • Try selecting a different pharmacy
  • If issue persists, print prescription and fax/call pharmacy

Can't Find Medication in Search

  • Try searching by generic name instead of brand
  • Check spelling
  • Search by NDC code if available
  • For compounded medications, use "Custom Medication" option

Drug Interaction Alert Won't Clear

  • Review the interaction details carefully
  • For critical alerts, you must provide clinical justification
  • Consider alternative medications
  • Consult pharmacist if unsure

Best Practices

üí° ePrescribe Best Practices:
  • Review Allergies First: Always check patient allergies before prescribing
  • Link to Diagnosis: Always link prescriptions to diagnoses for better documentation
  • Clear Instructions: Write patient-friendly SIG instructions
  • Check Interactions: Never ignore drug interaction warnings
  • Verify Pharmacy: Confirm pharmacy selection before submitting
  • Document Overrides: Always document clinical reasoning for alert overrides
  • Update Medication List: Keep active medications list current
  • Follow Up: Check prescription status to ensure pharmacy received it
